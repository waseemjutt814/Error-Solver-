# 🌐 Services Directory

External connections, API fetchers, Database queries, and third-party integrations go here.

## How to use the Numbered Pipeline here:
Services usually fetch data (`START`) or send data (`END`), making them perfect endpoints for your Error Solver graph.

### Examples:
- `010_FetchUser_STARTto011.ts`
- `060_SaveOrder_050toEND.ts`
- `900_SendEmail_800toEND.ts`
