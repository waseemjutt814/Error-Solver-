# 🛠️ Utils Directory

Helper functions, formatters, constants, and pure logic functions.

## How to use the Numbered Pipeline here:
Utils are usually small and imported everywhere, so they almost always Output to `MULT`. Sometimes they Input from `MULT` if they process different things.

### Examples:
- `500_FormatDate_MULTtoMULT.ts`
- `501_CalculateTotal_MULTtoMULT.ts`
- `001_Constants_STARTtoMULT.ts`
