# 📄 Pages Directory

Screen-level components (e.g., Next.js pages or React Router views) belong here.

## How to use the Numbered Pipeline here:
Pages are usually the grand orchestrators. They take `MULT` inputs from components/hooks and often output to `END` (rendering to the user).

### Examples:
- `001_HomePage_STARTtoEND.tsx`
- `200_DashboardPage_MULTtoEND.tsx`
- `404_NotFound_STARTtoEND.tsx`
