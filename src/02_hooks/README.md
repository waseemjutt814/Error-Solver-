# 🎣 Hooks Directory

Custom React Hooks (`useAuth`, `useFetch`, etc.) go here.

## How to use the Numbered Pipeline here:
Hooks represent internal logic flows. Map them to show how data is requested and processed in your app.

### Examples:
- `100_useAuth_011to101.ts` (Takes input from AuthForm)
- `101_useSession_100toMULT.ts` (Provides session data to multiple components)
